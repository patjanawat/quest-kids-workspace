# Run Commands — quest-kids

## Start Dev Server

| Mode | Command |
|------|---------|
| Default (QR / tunnel) | `npx expo start --clear` |
| Web browser | `npx expo start --web --clear` |
| Android emulator | `npx expo start --android --clear` |
| iOS simulator | `npx expo start --ios --clear` |
